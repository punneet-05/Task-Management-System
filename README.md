# Task Management System

## Tech Stack
- Node.js, TypeScript, Prisma, PostgreSQL
- Next.js App Router

## Features
- JWT Authentication (Access + Refresh)
- Task CRUD
- Pagination, Search, Filter
- Secure APIs

## Setup

### Backend
cd backend
npm install
npx prisma migrate dev
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## Note
Backend implementation is partially complete due to time constraints.
